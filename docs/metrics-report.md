# AI 质量指标说明
## 指标口径

| 指标 | 口径 | 面试解释 |
| --- | --- | --- |
| Top5 命中率 | 正确依据是否进入最终候选 | 衡量检索召回是否把正确材料送到模型前 |
| 引用准确率 | 引用是否能支撑回答 | 衡量回答是否可追溯，而不是只看文字像不像 |
| 拒答准确率 | 应拒答题是否拒答 | 衡量系统是否能控制幻觉和超范围请求 |
| 权限隔离通过率 | 权限/无权题是否正确处理 | 衡量无权资料是否被前置隔离 |
| 表格数值准确率 | 数值题是否输出正确值且避开禁用噪声 | 衡量 PDF 表格解析和表格 RAG 是否可靠 |
| 完整答案通过率 | 应回答题是否得满分 | 综合衡量内容、证据和格式质量 |

## 使用方式

1. 在 `/evaluation` 导入或维护题库。
2. 给题目补充场景、预期行为、账号和标准答案。
3. 运行评测，系统回填 Top5、引用、拒答和答案得分。
4. 用质量控制面板查看产品指标。
5. 根据主要失败原因决定下一轮优化优先级。

## 面试表达

我没有把 AI 能力当成一次性功能交付，而是建立了质量闭环。每次修改检索、表格解析或回答模板后，都可以用同一套题库回归，判断是否提升了准确率、是否引入权限泄露或拒答退化。
