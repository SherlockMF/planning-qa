## 两条贯穿全程的「黄金线」
- **验收标准线**：PRD 的每条验收标准（AC-1~AC-6）→ 在 evaluation-plan 的场景矩阵中都有对应项。
- **可演示线**：无 AI Key 时用内置 mock 全链路跑通，保证任何阶段都能即时演示。
